# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-17

### Added
- Initial MVP release
- Device detection via ADB and fastboot
- Support for Pixel 3a, OnePlus 6, and Surface Duo
- Bootloader unlocking wizard
- postmarketOS installation support
- Real-time terminal output overlay
- Progress tracking for download, decompress, and flash operations
